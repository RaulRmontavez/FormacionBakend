git init (Iniciar repositorio)
git remote add origin https:...(Añadir repositorio remoto)
git log (Para ver los commit del repositorio)
git clone https... (Clonar repositorio)
git status (Ver estado del git y nombre de la rama)
git diff (Ver la diferencia de los archivos)
git add nombrefichero (Agregar archivo)
git add -A (Agregar todos los archivos)
git add . (Agrega los archivos modificados)
git commit -m "Comentario sobre commit"(Para comitear todos los ficheros y añadirle un comentario)
git push origin RamaEnLaQueEstes (Subir ficheros)
git pull origin RamaEnLaQueEstes (Descargar ficheros)
git branch -a (Para ver todas las ramas)
git branch nombrerama (Para crear una rama)
git checkout nombrerama (Cambiar a la rama)
git reset (Eliminar todos los add hechos)
git --merge (ver rama mergeadas)
git merge nuevarama (Mergear otra rama)
git push origin --delete nombrerama (Borrar rama del repositorio remoto)
git branch -d nombrerama (Borrar rama local)
git branch -M nombrerama(Cambiar a una rama que no tiene nada)
git rm -r --cached (Nombre archivo) Borrar un archivo del repositorio
